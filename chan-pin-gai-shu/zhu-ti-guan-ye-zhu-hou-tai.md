# 功能摘要

| **功能模块** | **主要功能** | **优先级** |
| :--- | :--- | :--- |
| 概览 | 销售数据查看 | 高 |
| 概览 | 电竞服务 销售状态了解 | 中 |
| 概览 | 销售款项提现 | 高 |
| 主题店管理 | 主题馆信息设置 | 高 |
| 主题馆认证 | 后期自主入驻考虑，前期主要以平台大后台添加为主 | 低 |
| 管理员设置 | 电竞教练/店员权限分配 | 高 |
| 电竞服务 | 针对 电竞服务 与 电竞活动进行发布与修改 | 高 |
| 数据统计 | 简单的时间区间数据分析 | 低 |
| 订单管理 | 查看电竞服务销售订单 | 高 |
| 扫码核销 | 电竞 服务人员，进行课程订单与活动核销 | 高 |
| 会员 | 主题馆用户留存，联系与活动/赛事推介 | 高 |
